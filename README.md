# ERP-Management-System (Java,Swing,MySQL,Maven)
1) Designed and implemented a modular ERP system in Java using MVC architecture for scalability and maintainability.
2) Developed a user-friendly GUI with Java Swing and integrated a secure, scalable MySQL database.
3) Managed dependencies via Maven, including JDBC and external libraries.
4) Applied core OOP principles and implemented secure authentication with password hashing and account lockout.
5) Added advanced features like Backup/Restore, Import/Export, and role-based access (Admin, Student, Instructor) with maintenance mode controls.
