# ERP-Management-System (Java,Swing,MySQL,Maven)
1) Designed and implemented a modular ERP system in Java using MVC architecture for scalability and maintainability.
2) Developed a user-friendly GUI with Java Swing and integrated a secure, scalable MySQL database.
3) Managed dependencies via Maven, including JDBC and external libraries.
4) Applied core OOP principles and implemented secure authentication with password hashing and account lockout.
5) Added advanced features like Backup/Restore, Import/Export, and role-based access (Admin, Student, Instructor) with maintenance mode controls.

##How to Run

- For detailed setup instructions, database initialization steps, and complete login credentials, please refer to the README_ERP.pdf documentation or the attached demo video, which demonstrate the full installation, configuration, and execution workflow of the ERP Management System. The documentation also includes database schema details, system architecture, and example user credentials for testing purposes. 

##Key Features

01) Role-based authentication system (Admin, Instructor, Student)
02) Secure login with password hashing and account lockout after failed attempts
03) Automated database setup and SQL seeding using setup utility
04) Course catalog browsing and section-based enrollment system
05) Student course registration, drop functionality, timetable, and grade viewing
06) Instructor dashboard for managing sections, assigning marks, grading slabs, and statistics
07) Admin panel for managing users, courses, sections, and system settings
08) ERP maintenance mode that temporarily disables enrollment and grading actions
09) CSV/PDF transcript export and grade export functionality
10) Backup and restore functionality for system data
11) Data integrity checks preventing duplicate enrollments and unauthorized edits
12) Role-restricted access ensuring users can view and modify only authorized data
13) Comprehensive functional, edge-case, security, and performance testing coverage
